# Real-Time Indoor Thermal Comfort Monitoring and Analysis Using IoT Sensors

## Overview
This project focuses on monitoring indoor thermal comfort using IoT sensors and ESP32. 
The system collects environmental data such as temperature, humidity, and air quality.

## Objectives
- Monitor indoor environmental conditions
- Analyze thermal comfort levels
- Provide real-time sensor data

## Hardware Components
- ESP32 Dev Board
- DHT22 Temperature and Humidity Sensor
- DS18B20 Temperature Sensor
- MQ-135 Air Quality Sensor
- Breadboard
- Jumper Wires

## Working Principle
Sensors collect environmental data which is processed by the ESP32 microcontroller.  
The data can be monitored in real time and used to analyze indoor thermal comfort.

## Future Scope
- Cloud data storage
- Web dashboard visualization
- AI-based thermal comfort prediction
